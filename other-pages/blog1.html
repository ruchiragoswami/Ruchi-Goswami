File Naming Conventions for Web Developers

November 2019

While working in a regular desktop environment using an operating system like Windows or MacOS one can get away with using any file names. For example, "Some File Name".

In web development, server computers are utlized and the end users of any websites have to navigate to that website in their browsers. For this reason, the file naming convetions in web development are sliglty strict.

In Web Development, file names usually follow these rules:

  • Use alpha-numeric characters only
  • Use only lower-case characters
  • To separate words, use "-" or "_". Don't use spaces, dots or slashes
