Difference between <strong> and <b> tags in HTML

August 2021

While writing content in the HTML, we often come across cases where we have to use bold fonts. Reasons for using bold texts can be different. Most of the times, we just want to draw attention of the users. Sometimes, the text itself has more importance that the surrounding text and therefore, we need to make it bold.

Using <strong> tag

The bolded text has more importance than the surrounding text. For exmaple, "This text is <strong> important </strong> ! And this text is not important. "

Using <b> tag

Use <b> tag when you just want to draw the user's attention, or to make the text appear logically different or aesthctically pleasing.
